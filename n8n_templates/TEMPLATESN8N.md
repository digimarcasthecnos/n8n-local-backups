Relatório de Projeto: Extração em Massa de Templates n8n1. Visão GeralO objetivo deste projeto é a extração e armazenamento local de toda a biblioteca de templates do n8n (aprox. 9.600 fluxos) para permitir a busca, modificação e estudo de automações diretamente no computador, utilizando o Visual Studio Code como interface principal.2. Stack UtilizadaEditor: Visual Studio Code (VS Code).Linguagem: Python 3.14 (Ambiente Anaconda).Versão do n8n: 2.19.4 (Instância Cloud).Protocolo de Versão: Git (proposto para extração via repositório).Ferramentas Auxiliares: DownGit (testado e descartado por instabilidade).3. Decisões Tomadas e HistóricoTentativa 1: n8n-atom (VS Code): Decidimos não utilizar para a extração massiva, pois a extensão sincroniza apenas workflows já existentes na conta do usuário, não sendo capaz de "puxar" a biblioteca pública de 9.000 itens de uma só vez.Tentativa 2: Web Scraping via Python: Foi desenvolvido um script em Python para consumir a API do n8n.io. A decisão foi abandonada após o erro JSONDecodeError, causado por proteções de acesso (Cloudflare) que bloqueiam requisições automatizadas sem chaves de parceiro.Tentativa 3: Extração de Pasta Específica (DownGit): Tentamos baixar apenas a pasta de assets do GitHub do n8n. O site DownGit apresentou erro Invalid URL e undefined, provavelmente devido a mudanças recentes na estrutura de permissões do GitHub.Decisão Final: Download do repositório completo (master.zip) diretamente do código-fonte do n8n. Esta é a única forma garantida de obter os arquivos JSON brutos sem restrições de API ou bloqueios de rede.4. Convenções AdotadasLocalização dos Arquivos: Os templates oficiais residem no caminho interno: packages/cli/assets/templates.Formato: Todos os templates devem ser mantidos em formato .json para compatibilidade com o comando de importação do n8n.Busca Global: Utilização do atalho Ctrl + Shift + F no VS Code para indexar e pesquisar funcionalidades dentro dos milhares de arquivos baixados.5. Próximos PassosConclusão do Download: Aguardar o término do download do n8n-master.zip.Extração Cirúrgica: Localizar a pasta templates dentro do ZIP e movê-la para a pasta de trabalho n8n_templates_master.Configuração de Credenciais: Conforme discutido, ao abrir qualquer JSON extraído no n8n, será necessário vincular as APIs pessoais (OpenAI, Google, etc.), pois os templates vêm sem chaves de acesso por segurança.Limpeza de Ambiente: Remover os scripts Python que falharam (extrair_n8n.py) para evitar confusão no diretório.
